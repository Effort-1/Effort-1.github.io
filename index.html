让git保存用户名和密码
发表于2021-04-21
1找到 .gitconfig 文件.gitconfig 配置文件一般在 C>user>（你的用户名）文件夹下 如果实在找不到这个文件，就使用这个软件 Everything 去搜索 附上链接 百度云 提取码：6666 在此目录下打开 git Bash 运行： git config --global credential.helper store .gitconfig 文件下就会多了下面的内容 同时该目录下会生成一个 .git-credentials 文件，里面保存的有你的用户名和密码 当你再次向 github 提交时你只需要输入一次，后面在提交时就不在需要重新输入 tips: 后面提交依然会弹出 GitHub 用户登录UI窗口，直接点取消就可以了
